1. User Authentication

- User login
- User registration
- Role based access

- Add product

- Delete product
- View product list

3. Inventory Management

- Track stock quantity

- Search products
- Add products to cart
- Calculate total
- Generate bill

5. Sales Dashboard

- View total sales
- View daily orders
- Product performance
